JS Game Server

Sandboxed JavaScript Game Server

View project onGitHub

The proposal game server for the js13kGames Competition Competition.

This project is a important piece of the proposal posted on a html5gamedevs forum topic. An effort maintain the server category on the competition.

You can help! Test it.

Deploy

$ git clone git://github.com/aurium/js-game-server.git
$ cd js-game-server
$ npm install
$ npm test

Runing

You must to set the GAMES_DIR environment variable to the server.

On BASH-like consoles you have a pretty simple one-line command:

$ cd /path/to/js-game-server
$ GAMES_DIR=/the/dir/with/games npm start

Try it with the game examples:

$ GAMES_DIR=examples npm start
